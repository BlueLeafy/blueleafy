For this project, I developed a dynamic date-time picker with an integrated dark mode toggle. The goal was to create a user-friendly component that enhances user interaction through intuitive design and modern web features.

I started by building the basic structure using HTML5, ensuring semantic and accessible code. This included a date-time input field that users can easily interact with. The design was crafted using CSS3 for a clean and responsive layout. I created separate stylesheets for the default and dark themes (styles.css and dark-styles.css), ensuring the picker looks great across different devices and lighting preferences.

To make the picker dynamic, I used JavaScript. This script handles the functionality of the date-time picker, responding to user inputs and ensuring smooth transitions between light and dark modes. I implemented a dark mode toggle using additional JavaScript (darkTheme.js). The user can switch between light and dark themes with a button click, which enhances usability, especially in low-light environments. The entire component is fully responsive, adapting to different screen sizes and ensuring a consistent user experience on both desktop and mobile devices.

HTML5 was used to structure the page and form elements. CSS3 provided responsive design and theming, creating separate styles for light and dark modes. JavaScript powered the interactive elements, including the dynamic date-time picker and theme switching.
